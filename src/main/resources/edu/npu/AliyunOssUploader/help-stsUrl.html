Your Alibaba Cloud OSS temporary authorization URL
STS is a temporary authorization mechanism provided by Alibaba Cloud. Through STS, a temporary access credential can be issued to third-party users (such as sub-accounts, temporary users, etc.) to access OSS. The use of STS can effectively protect the security of your OSS data.
The STS URL is a temporary authorization URL obtained through the STS service, which contains temporary credential information for accessing OSS. The validity period of the STS URL is set by you, generally 15 minutes, and needs to be re-acquired after expiration.
If necessary, please refer to Alibaba Cloud's official description of STS.
