Full URL of your Alibaba Cloud OSS EndPoint
EndPoint is the access point for OSS services. EndPoints in different regions are different. For example, the EndPoint in Hangzhou is oss-cn-hangzhou.aliyuncs.com, while the EndPoint in Silicon Valley is oss-us-west-1.aliyuncs.com.
If your service is deployed in an intranet environment, you also need to add the internal suffix. For example, the intranet EndPoint in Hangzhou is oss-cn-hangzhou-internal.aliyuncs.com.
If necessary, you can refer to Alibaba Cloud's official description of endPoint.
